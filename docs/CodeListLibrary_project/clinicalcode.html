Get the permitted codelists. Codelists do not carry permissions. Components are permitted by virtue of being part of a permitted concept. There is a one-to-one relation- ship between codelists and components. If we want all the permitted codelists, we need to find all the permitted components and add each components codelist to the list. If we want to determine if codeListID is permitted, we just need to find the component that contains it. (work only on live version since it is used in edit form)
