Concept Library API

The concept library API allows programmatic interaction with code lists, so that they can be directly used in queries, statistical scripts, etc.

(Replace the number 0 in the url with the relevant concept/working set id.
Replace the number 1 in the url with the relevant version_id of the concept/working set.)

Concepts:

  • To export the code list of the latest version of a concept
    1. JSON

      {{ export_concept_codes }}?format=json

    2. XML

      {{ export_concept_codes }}?format=xml

  • To export the code list of a specific version of a concept
    1. JSON

      {{ export_concept_codes_byVersionID }}?format=json

    2. XML

      {{ export_concept_codes_byVersionID }}?format=xml

Working Sets:

  • To export the code list of the latest version of a working set
    1. JSON

      {{ export_workingset_codes }}?format=json

    2. XML

      {{ export_workingset_codes }}?format=xml

  • To export the code list of a specific version of a working set
    1. JSON

      {{ export_workingset_codes_byVersionID }}?format=json

    2. XML

      {{ export_workingset_codes_byVersionID }}?format=xml
